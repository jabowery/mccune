Installing mccune, Mace4, and Friends

Unix-like Systems

Here is a quick example for Unix-like systems, including Linux and Macintosh OS X. Visit the mccune Web page and download the current version of LADR. The filename should be something like LADR-June-2006A.tar.gz; make sure that file is in your current directory. Run the following commands. 
% zcat LADR-June-2006A.tar.gz | tar xvf -
% cd LADR-June-2006A
% make all

mccune, Mace4, Prooftrans, and several other programs should now be in the directory LADR-June-2006A/bin. You can either include that directory in your search path or copy those programs to some directory that is already in your search path.
